# Funda Çiçek Wedding

Wedding dress website for Funda Çiçek, located in Bursa, Turkey. Features custom-made wedding dresses, rentals, and sales.

## Features

- Hero slideshow with wedding dress images
- Gallery with hover tilt effects
- About section with owner information
- Contact section with location, phone, WhatsApp, and Instagram
- PWA Support (Installable)
- Responsive design

## Tech Stack

- Astro
- Tailwind CSS
- Vite PWA Plugin

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
